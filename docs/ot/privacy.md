# 数据隐私说明

----------

Usolia 致力于为您创造一个良好的游戏环境，同时确保您的个人隐私安全无虞。

Usolia 的所有信息存储服务器均位于欧盟境内。没有第三方可以访问这些服务器，Usolia 也不会将收集的信息分享给任何第三方。

下方列出了 Usolia 会记录并储存的数据：

#### 游戏服务器内：

- 玩家 UUID 与游戏名
- 您在注册时填写的邮箱
- 您登录时所使用的 IP 地址
- 您的公开及私人（msg/pm/email) 聊天记录
- 您执行的指令记录（离线登录时的 /login 和 /register 指令除外）
- 其他插件产生的日志文件

#### 官网论坛：

- 您注册使用的名称
- 您的邮箱地址
- 您在执行操作时使用的 IP 地址
- 您在论坛留下的其他个人信息，例如生日、地址等

----

### 数据保留期限：

- 游戏服务器内：
	- 账号有关数据：长期，直至账号注销
	- 游戏内操作记录（例如方块放置/破坏，游戏货币交易记录等）：2-5 年
	- 其他数据（例如聊天记录）：长期
- 官网论坛：长期，直至账号注销

----

### 请求数据删除：

你可以向我们申请删除你的游戏服务器或/和论坛账号。

#### 游戏服务器账号

向我们的服务邮箱 support@usolia.net 发送邮件，邮件主题为“申请账号数据删除”，内容包含你的游戏账号的**游戏名**，并在邮件结尾写下“**我确认相关数据一旦删除将不可恢复**”。

收到请求后，我们会确认你的确是账号的所有者。身份确认后，我们会将你的账号删除。

?> 出于安全考虑，某些数据依然会保留在数据库中。

#### 论坛账号

访问 [论坛 > 我的账号 > 注销账号](https://usolia.net/account/delete)，并按照页面提示进行操作。

?> 在提交申请后，你的账号将进入为期 14 天的注销等待期，在该段时间内你可以随时 [访问论坛](https://usolia.net) 并登录你的账号以撤销注销请求。14 天过后，你将无法再访问你的账号。

?> 账号删除过程只会删除你的个人信息，你此前发布的所有内容在去除你的相关信息后仍将保留。
