# 处罚机制

本文旨在为服务器的处罚机制运作提供一定的透明度。管理团队始终保留所有权利。

---

## 禁言

滥用游戏内聊天栏、服务器QQ群、Discord服务器 等官方交流平台。

行为失当者也有可能被跨平台禁言。

每次禁言会附带一次服务器警告。

## 警告

当失当行为不足以给予封禁处罚时，会给予警告处罚。

警告次数达到 3 次后，所有本应为警告的处罚措施会改为封禁。

## 封禁

行为失当者可能会被封禁几天-几个月（期间不允许使用其他账号登录），最高至永久封禁（允许使用其他账号重新注册）或永久驱逐（根据特征进行封禁，不允许再登录服务器）。

封禁力度取决于多个方面，包括但不限于失当行为严重性、过往封禁记录和影响范围。
