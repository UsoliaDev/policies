# 社区准则

我们希望营造一个温暖的社区环境和愉快的游戏氛围，让新成员可以没有顾虑的吹水聊天，老玩家也愿意时常冒泡并 ~~挑逗新人~~ 与新成员友好互动。因此请认真阅读以下内容，帮助我们共同维护一个和谐的社区环境。

---

## 前置规则

- [ Usolia Network 服务条款](https://usolia.net/help/terms/)

## 适用范围

Usolia Network 社区

## 细则

对于未涉及的失当行为，管理团队有权根据实际情况做出反应。

#### 人身攻击 (1.1)

包括但不限于：

- 侮辱、谩骂、恐吓、胁迫、挑衅或歧视他人
- 对他人开过度玩笑，或在他人明确提出反对后仍不停止
- 在没有确切证据指向的情况下，以恶意揣测他人

#### 擦边球行为 (1.2)

请勿试探服务器或他人的底线。

#### 造谣、传谣以及煽动 (1.3)

制作、传播包含不实内容的信息，或有意在内容中包含情绪化内容。

#### 公开他人隐私 (1.4)

在未得到对方明确许可的前提下，公开对方的个人信息。

#### 屡次违反规则 (1.5)

多次违反社区准则或其他规则，或做出规则中未提及的失当行为。

#### 曲解规则 (1.6)

钻牛角尖、玩文字游戏等意图为曲解利用规则的行为。

#### 功能滥用 (1.7)

滥用服务器提供的各项功能的行为。

#### 广告与垃圾信息 (1.8)

例如在社区内发布某服务器提供商的广告。

#### 对与 Usolia 具有竞争关系的实体的提及 (1.9)

允许基于讨论或研究等此类目的，在 Usolia 社区内提及与我们具有竞争关系的实体（例如其他 Minecraft 服务器）。

以宣传和引流为目的的提及不被允许。

#### 政治宗教与新闻时事 (1.10)

此类内容经常会导致争执与矛盾，在QQ群等对此类信息较为敏感的平台还可能导致平台封禁，故管理团队在发现此类信息后可能会直接移除，并禁言或封禁此类内容的发布者。

#### 消极情绪待人处事或其传播 (1.11)

如果长期以消极情绪待人处事被其他社区成员多次反映，或在日常公开散布消极情绪，可能会导致处罚。

#### 假冒管理团队成员 (1.12)

使用具有误导意义的头像和昵称或以其他方式，试图让其他人认为自己是管理团队成员的行为。

#### 请勿对小事上纲上线 (2.1)

友好环境得来不易，锱铢必较只会适得其反。

#### 请勿频繁 @ / ping 管理团队成员 (2.2)

如果一直有人 @ / ping 管理员，ta 可能会因为设备一直振动/响铃提醒从而关掉通知，这样在遇到真正需要及时查看处理的消息时就可能错过。

~~管理团队成员都有着猫一般的好奇心~~，一般会经常浏览群消息记录；其他玩家在看到消息后也可能能解决相关问题，因此没必要一直 @。

#### 在正确的位置提交问题 (2.3)

服务器论坛针对不同类别的问题有很多专门的处理板块：

* 一般性问题问询：[uso.li/help](https://uso.li/help)
* 建议与意见反馈：[uso.li/ideas](https://uso.li/ideas)
* 服务器错误与漏洞反馈：[uso.li/bug](https://uso.li/bug)
* 违规玩家举报：[uso.li/report](https://uso.li/report)
* 申诉：[uso.li/appeal](https://uso.li/appeal)

如果某件事需要你用50个字以上来描述，就可以考虑把它提交在论坛上了。

而对于只是主城少了某个玻璃板，或者商店某个物品突然不自动补货了这种事，在群里@一位管理就可以了。

#### 心理状态与消极情绪有关 (2.4)

我们了解这种情况；但有些地方并不是解决这类问题或发泄的合适场所，如果服务器交流平台上频繁出现类似内容，整体气氛也会受到影响。

你可以选择与可以在现实中对你产生影响的人，例如你值得信任的朋友联系，很多情况下这比从线上获得的帮助带来的效果更佳。

如果情况趋于紧急，你也可以借助医疗力量进行干预：
